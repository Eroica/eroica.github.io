Here you can get a glimpse on the kind of work I do and which tools or languages I use.

A Bisect modifier for Blender. You can use it to bisect large meshes non-destructively, e.g. to create animations.

Using Booleans to bisect a mesh can freeze Blender indefinitely for large meshes (e.g. 30+ mio vertices), or is error-prone in non-manifold areas. The alternative Mirror and Mask modifier combination does not allow you to close the bisected mesh (the open face), and requires creating Vertex groups in Edit mode which can also take a long time for huge meshes.

Paletti is a small program that “posterizes” an image into a smaller amount of colors and shapes. This way you can extract primary colors or simplify an image artistically. It uses Leptonica for the color quantization.

The first version was written in Vala using Leptonica’s C API directly by a custom VAPI file. When I switched to Windows, I re-wrote Paletti in Kotlin using JavaFX and a custom C++ backend to call Leptonica from JNI. Paletti now even has an Android version which uses the same C++ backend.

For the JavaFX version, I paid special attention to make it look like a modern Windows application. It features client-side decoration in a combined header bar (similar to its Vala/GTK roots) where I minutely reconstructed WinUI 3 widgets, including hover effects and animations.

An online mesh viewer for 3D segmentations. It is designed and built by me using three.js. Depending on the project, I also post-process and/or retopologize the meshes to improve visuals while keeping memory consumption low.

I made a small contribution to LuaFileSystem which improved Windows support.
