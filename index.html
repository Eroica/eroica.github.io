Here are some things that I worked on that might be useful to you.

{% include feather/triangle.svg %}

I wrote a Bisect modifier for Blender to work with large meshes at work. We often want to show and animate the inner parts of cells and other organelles, but Blender currently (June 2021) only supports bisecting in Edit mode or with a Mirror and Mask modifier combination. However, our meshes can be made up from around 30 mio vertices so that one cannot simply enter Edit mode or add Vertex groups. For the same reason, the usual approach of using Booleans can freeze Blender indefinitely.

The modifier can also close the bisected, open face (for manifold meshes), and can set another material for this area (similar to Booleans).

{% include feather/star.svg %}

Paletti simplifies an image into smaller amount of colors. After dragging an image into Paletti, you can customize the amount of colors with the slider. I initially wrote it to generate a color palette off an image, but now it’s more accurate to describe it as doing “posterization” using Leptonica. There is a small difference in which colors are yielded in this process.

The first version was written in Vala using Leptonica’s C API directly by a self-written Vapi file. When I switched to Windows, I re-wrote it in Kotlin using JavaFX and a custom C++ backend to call Leptonica from JNI. Paletti now even has a (work-in-progress) Android version which makes use of the same C++ backend.

I paid special attention to make it look like a modern Windows application instead of using the default JavaFX style, going as far as implementing client-side decoration (CSD) in a combined header bar (similar to it’s original Vala/GTK roots) which minutely imitates hover effects, focus areas for accessibility, and more.
