Here are some things that I wrote that might be useful to you.

{% include feather/triangle.svg %}

I wrote a Bisect modifier for Blender which allows you to bisect large meshes non-destructively. At work, we often want to show and animate inner parts of biological cells, but Blender currently (September 2021) only supports bisecting in Edit mode or with a Mirror and Mask modifier combination. However, our meshes can be made up from around 30 mio vertices so that entering Edit mode, adding Vertex groups, or using Booleans on them can freeze Blender indefinitely.

With a modifier, you can easily create an animation that bisects a mesh frame by frame. It can also close the bisected, open face (for manifold meshes), and can set another material for this area (similar to Booleans)

{% include feather/star.svg %}

Paletti simplifies an image into a smaller amount of colors. I initially wrote it to generate a color palette off an image, but now it’s more accurate to describe it as doing “posterization” using Leptonica. There is a small difference in which colors are yielded in this process.

The first version was written in Vala using Leptonica’s C API directly by a self-written Vapi file. When I switched to Windows, I re-wrote Paletti in Kotlin using JavaFX and a custom C++ backend to call Leptonica from JNI. Paletti now even has a (work-in-progress) Android version which makes use of the same C++ backend.

I paid special attention to make it look like a modern Windows application instead of using the default JavaFX style, going as far as implementing client-side decoration (CSD) in a combined header bar (similar to it’s original Vala/GTK roots), minutely imitating hover effects, and adding custom focus areas for improved accessibility.
