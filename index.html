Here are some examples of the kind of work I do:

{% include feather/star.svg %} Merit Cards, a new flashcard application for Windows. I use it to study languages because I don’t really like how Anki works. A note is made up of individual cards (“facets”) and are scheduled after a basic spaced-repetition algorithm. You can also schedule your notes freely just like traditional Leitner learning boxes. Kotlin JavaFX SQLite

My rendering of a T cell and cancer cell was featured on the cover of Science, Issue 6591, on April 2022. The meshes were obtained from an electron microscope dataset as part of Alex Ritter’s research about how T cells destroy cancer cells.

An online mesh viewer for 3D segmentations (here used with example meshes). Compared to Google’s <model-viewer>, it is modelled after an ECS architecture which can dynamically switch between interaction and pause (completely skipping draw calls, thus saving power). JavaScript three.js KnockoutJS

A Bisect modifier for Blender. It allows you to bisect large meshes (30+ mio vertices) non-destructively, e.g. for creating animations. C++

Paletti is a little program that “posterizes” an image into a smaller amount of colors and shapes. This way you can extract primary colors or simplify an image artistically. It uses Leptonica to process the image. Kotlin JavaFX SQLite C++ Vala GTK Android

I made a small contribution to LuaFileSystem which improved Windows support. Lua C
