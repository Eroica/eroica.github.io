Here are some examples of the kind of work I do:

My rendering of a T cell and cancer cell was featured on the cover of Science, Issue 6591, on April 2022. The meshes were obtained from an electron microscope dataset as part of Alex Ritter’s research. He is a Cell biologist at Genentech studying how T cells destroy cancer cells.

It was one of my most interesting projects in scientific visualization so far, and I am very thankful for getting the chance to work on this.

Merit Cards is my latest side project. It is a flashcard application that I use to study languages.

There are many similar programs, but I created Merit Cards for my way of collecting and studying notes: There are no “Front” or “Back” fields that you add values to and for which you create card templates like in Anki. Instead, a note simply consists of a stack of index cards (“Facets”) that you study one-by-one. The spaced-repetition algorithm is very simple, although this hopefully makes it more approachable than Anki’s.

Like Paletti, Merit Cards is written in Kotlin using JavaFX and my WinUI 3 theme. It is currently still very basic, but since I rely on it myself I’m gradually improving it!

A Bisect modifier for Blender. You can use it to bisect large meshes non-destructively, e.g. for creating animations.

Using Booleans to bisect a mesh can freeze Blender indefinitely for large meshes (around 30+ mio vertices), or is error-prone in non-manifold areas. The alternative Mirror and Mask modifier combination does not allow you to close the bisected mesh (the open face), and requires creating Vertex groups in Edit mode which can also take a long time for huge meshes.

Paletti is a small program that “posterizes” an image into a smaller amount of colors and shapes. This way you can extract primary colors or simplify an image artistically. It uses Leptonica to process the image.

The first version was written in Vala using Leptonica’s C API directly with a custom VAPI file. When I switched to Windows, I re-wrote Paletti in Kotlin using JavaFX and a custom C++ backend to call Leptonica from JNI. There is now even an Android version which uses the same C++ backend.

For the JavaFX version, I paid special attention to make it look like a modern Windows application. All widgets are minuite reconstructions of WinUI 3 controls, including hover effects and animations. In a recent build for Windows 11, Paletti also features a Mica-styled window chrome by calling native Windows functions from JavaFX.

At my employer I built an online mesh viewer for 3D segmentations using three.js. Compared to Google’s <model-viewer>, it is modelled after an ECS architecture which can dynamically switch between interaction and pause (completely skipping draw calls, thus saving power).

Depending on the project, my work also includes post-processing, manual retopology, or texturing to improve visuals while keeping memory consumption low.

I made a small contribution to LuaFileSystem which improved Windows support.
