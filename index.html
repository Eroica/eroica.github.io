Software & visualization projects

I’m @Eroica on GitHub and some other places. I work as a web developer and also on a few software and 3D visualization projects.

Software

{% include feather/star.svg %}Merit Cards — a new flashcard application for Windows. I use it to study languages because I needed something different to Anki. A note is made up of individual cards (“facets”) and are scheduled after a basic spaced-repetition algorithm. You can also schedule your notes freely just like traditional Leitner learning boxes.

Paletti — a little program that “posterizes” an image into a smaller amount of colors and shapes. This way you can extract primary colors or simplify an image artistically. It uses Leptonica to process the image.

Libraries

I published the WinUI-inspired theme from my JavaFX applications as a separate library: javafx-fluent-theme.

Plugins/Extensions

A Bisect modifier for Blender — it allows bisecting large meshes (30+ mio vertices) non-destructively, e.g. for creating animations.

3D Visualization

Scientific visualizations — for example, the front cover of Science, Volume 376, Issue 6591, April 22, 2022. I created more than 200 renderings and animations while working at ariadne.ai.

An online mesh viewer for 3D segmentations (here with example meshes) — compared to Google’s <model-viewer>, it is modelled after an ECS architecture which can dynamically switch between interaction and pause (completely skipping draw calls, thus saving power).
